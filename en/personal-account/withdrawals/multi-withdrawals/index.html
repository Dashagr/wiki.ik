Multi withdrawals to cards

If you need to withdraw funds to several cards at once, use the bulk withdrawal function and make a request for multiple withdrawals in just a few steps.

Multi withdrawal creation​

Step 1. Page fot multiple withdrawals to cards​

Go to the My chechouts page, find the checkout from which you want to withdraw money, and click on the special 'Card multi withdrawals' icon.

'Card multi withdrawals' icon

Step 2. Uploading a file with recipient data​

On the next page, download the .xlsx file with the card data of the recipients. If you are doing this for the first time, the template of .xlsx file is available. Fill it with data as shown in the template. (The first column is the amount, the second is the card number, and the third one – is the full recipient’s name).

Multi withdrawal page

Step 3. Data validation​

Double-check the data, automatically pulled from the file, for correctness. If you found an error, it can be corrected right in the field. There is also a function to cancel the output to the card of a specific recipient if his data was redundant in the file.

Validation of data before the withdrawal

If all the data is entered correctly, click on the 'Create withdrawal' button.

Congratulations! You have created a bulk withdrawal request. Each individual withdrawal request will be processed separately in turn. You will be able to find out about the formation of the request right away on the same window next to the recipient's card data.
