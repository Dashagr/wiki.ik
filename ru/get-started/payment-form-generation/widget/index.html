Виджет

Чтобы отправить плательщика с веб-сайта на страницу оплаты, вы можете разместить виджет в HTML коде страницы сразу рядом с товаром или услугой. Виджет – это HTML форма, использующая метод GET, и при нажатии на кнопку 'Рау' перенаправит плательщика на платежную страницу Interkassa. Сгенерировать HTML код виджета можно в личном кабинете в нескольких шагах.

Шаг 1. Перейдите на страницу создания платежной формы​

В личном кабинете перейдите на страницу 'Мои кассы'. Выберите кассу, для которой необходимо сгенерировать платежную форму и нажмите специальный значок 'Генерация платежной формы'.

Кнопка 'Генерация платежной формы'

Шаг 2. Заполните необходимые поля​

Теперь вы на странице создания формы для платежей. В полях формы укажите номер платежа, согласно вашей системе учета (если требуется), сумму и по желанию дополните платеж описанием.

Создание формы для платежей

информация

Доступна также функция еще большей кастомизации платежной формы, кликните на чекбокс 'Расширенная форма', откуда вы можете настроить углубленные параметры для платежной страницы. Например, вы можете отправлять дополнительные значения в колбеке или ответе по API для кастомной бизнес логики на вашей стороне.

Шаг 3. Создание платежной формы​

После того, как все данные заполнены, нажмите кнопку 'Создать Форму'.

Сгенерированный виджет для платежа

Готово! Виджет создан.

В форме используется метод GET, который при нажатии на кнопку перенаправит плательщика на платежную страницу Interkassa. Значение value внутри тега <input> отвечает за текст, отображаемый на кнопке (по умолчанию, на кнопке отображается значение 'Pay'). После встраивания виджета в веб-сайт можно кастомизировать его на вашей стороне, предоставив стили веб-сайта.

дополнительно

Также на платежную страницу можно добавить логотип и краткое описание вашей торговой точки. Подробная инструкция находится по ссылке.

Выставленные счета имеют разные статусы с момента создания до момента оплаты. Вы можете отслеживать статус счета на странице 'Принятые платежи'. Дополнительную информацию об отслеживании платежей см. в разделе 'Управление платежами'.
