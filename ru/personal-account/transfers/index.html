Переводы

Мерчанты могут осуществлять переводы средств между кошельками касс одного аккаунта или между кошельками касс из разных аккаунтов.

Внимание!

Эта операция не доступна по умолчанию. Чтобы ее активировать - обратитесь к своему менеджеру, который подключит функцию перевода для конкретной кассы.

Комиссия для переводов настраивается непосредственно под мерчанта и должна быть согласована с аккаунт-менеджером. Перевод денег происходит без конвертации, то есть в пределах одной валюты. Например, UAH → UAH, USD → USD и т.д.

Создание перевода:​

  • Шаг 1. Перейдите на вкладку 'Переводы' из верхнего меню.

Вкладка 'Переводы'

  • Шаг 2. Нажмите на кнопку 'Создать перевод'.

Главная страница раздела 'Переводы'

  • Шаг 3. Заполните соответствующие поля формы. Id кассы получателя можно узнать со страницы с перечнем всех касс сразу под названием, или спросить у адресанта, если этот перевод осуществляется между аккаунтами. Расчет производится автоматически и фактическая сумма зачисления отображается сразу в поле сумма к получению.

Форма создания перевода

  • Шаг 4. Нажмите на кнопку 'Создать перевод'.

  • Шаг 5. Если настроена СМС авторизация, подтвердите операцию кодом.

Форма подтверждения

  • Готово! Увидеть, что трансфер состоялся, можно на вкладке 'Переводы' в верхнем меню.
подсказка

Также можно экспортировать данные трансфера в excel формате со страницы 'Мои кассы'.

FAQ​

Есть лимиты на количество переводов?
Нет, вы можете проводить столько переводов в день, сколько нужно. Лимитов на сумму перевода тоже нет.
Сколько времени занимает перевод?
Эта операция мгновенная, поэтому вы сразу можете увидеть ее результат на вкладке 'Переводы'.
Можно ли внести изменения в уже состоявшийся перевод?
Нет, к сожалению, такая возможность отсутствует.
Нужно ли активировать функцию перевода для каждой кассы отдельно?
Так, чтобы перевод был успешен, менеджер должен включить в кассы соответствующую функцию.
Для каких валют можно сделать перевод?
На странице список валют. Вы можете узнать все валюты, которые поддерживает Interkassa. Предложения возможны для каждой из них.
