Создание и настройка кассы

Сразу после регистрации в личном кабинете Вы можете создать кассу для приема платежей. Касса в нашей системе представляет собой совокупность счетов (кошельков), на которые будут поступать средства плательщиков.

К одной кассе могут быть привязаны кошельки в разных валютах, которыми Вы пользуетесь при выставлении счетов плательщикам. Если вы выставляете счет в одной из валют, например в USD, то после оплаты кошелек в выбранной кассе будет пополнен этой же валютой.

На кассу средства поступают в валюте, в которой был создан счет, но плательщик может рассчитаться с помощью любого из платежных методов, доступных для этой кассы. К примеру, если валюта кассы USD, а клиент оплатит свой товар в EUR, сумма будет конвертирована и на кассу зачислятся USD. Текущий курс конвертации валют можно посмотреть с URL: https://api.interkassa.com/v1/currency.

Для удобства управления трафиком и мониторинга данных, можно создать несколько касс для одного веб-сайта.

Создание кассы​

Шаг 1. Войдите в личный аккаунт.​

Перейдите на страницу входа и введите свой логин и пароль.

Шаг 2. Заполните форму​

На главном экране кликните на кнопку 'Создать кассу' и заполните форму следующей информацией:

  • Тип кассы: Персональная (если касса необходима для стандартной работы в режиме приема платежей) или Агентская (если хотите получать награду за подключенных вами мерчантов).
  • Адрес сайта: URL сайта, который будет принимать онлайн-платежи.
  • Название кассы: как вам удобно. Если касс будет несколько, то их будет проще находить.
  • Валюта кассы: UAH (эта валюта доступна по умолчанию. Другие валюты, которые вы обговорили с менеджером, будут доступны позже -- их подключат на стороне Interkassa).

Создание кассы

Шаг 3. Нажмите на кнопку 'Создать'.​

Готово! Касса создана и подтверждена. Теперь можно приступать к генерации платежной формы.
