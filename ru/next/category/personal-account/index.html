Работа в личном кабинете

Наша платформа предлагает пользовательский интерфейс для управления и настройки вашей учетной записи Interkassa. Вы можете использовать его для управления платежами, снятием средств, мониторинга вашей интеграции и т.д. В этом разделе вы найдете подробные туториалы для ознакомления с возможностями личного кабинета.

📄️ Процесс оплаты через Interkassa

Когда все этапы подключения успешно пройдены, вы сможете принимать оплату от клиентов и отслеживать свои доходы через личный кабинет.

📄️ Выставленные счета

Вы можете просмотреть информацию обо всех счетах, которые были созданы в вашей учетной записи. Для этого из бокового меню перейдите на страницу 'Issued invoices'. Эта страница содержит таблицу учетных записей, отсортированных по номеру учетной записи по умолчанию (самые последние находятся вверху).

📄️ Управление платежами

Вы можете просматривать информацию о всех платежах, поступивших на ваш аккаунт. Для этого из верхнего меню перейдите на страницу 'Операции' -> 'Принятые платежи'. На этой странице находится таблица с платежами, по умолчанию отсортированными по дате (самые новые сверху).
